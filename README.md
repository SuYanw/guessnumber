# Guess Number Game

This is a simple Guess number game in Elixir.
After download, you can running this game with
```
	mix run
```


Pick a difficult level (1,2 or 3):
1 - Level Easy ( 1 than 10 )
2 - Leve medium ( 1 than 100 )
3 - Level expert ( 1 than 1000 )
